Activity launches another Activity

So far, we worked with a single Activity in our application. But most of the times you will be using more than one Activity in your app. Either startActivity() or startActivityForResult() can be used to launch another Activity from your current Activity. When you use startActivity(), you wont get back results(sucess code or data) from the newly launched Activity. If you want to get back results, use startActivityForResult(). See the next sample on using startActivityForResult().

In this app we will be using startActivity(). Notice the use of setClass() for Intent created for launching another Activity. . This kind of Intent is called implicit Intent, and should be used if your new Activity is in the same Application. To learn more about Intents and implicit/explicit Intents see the reference docs for Intent .
