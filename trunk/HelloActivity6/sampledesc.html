Activity sends data to another Activity

There are a few ways to send data between Activities, most simplest of all is to use Bundle extras. Click source tabs to how the HelloActivity sends "Hello World!" string to HelloAnotherActivity. Apart from string you can send primitive data, serializable objects or parcelables.

Reference docs for various extras.putXXX()/getXXX() methods can be found here.

To read more about other options for sending data see framework FAQs. We will cover these methods in the samples ahead.
