Simple Activity with string values defined in xml

We will now extend our HelloActivity to use string values defined in res/values/strings.xml. Check out the strings.xml file. In our layout xml file we retreive this string value using syntax @string/id where id is the auto generated id for this string resource.

Defining these strings as resources instead of hardcoding them makes it easier to support alternative languages and configurations. To read more about how to do this see Resources and i18n .
